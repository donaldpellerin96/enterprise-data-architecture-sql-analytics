# Enterprise Data Architecture & SQL Analytics

## Project Overview
This project focused on designing and implementing an enterprise relational database system for a financial services organization. The objective was to develop a scalable data architecture supporting analytics, fraud monitoring, reporting, and customer account management.

---

## Business Problem
The organization required a structured enterprise database capable of managing customer, account, and transaction data while supporting operational reporting, financial analysis, and fraud detection workflows.
